# 试验243林晨
====================================================================
  RaspiClaw v3.0.5 — AI Agent 系统 使用安装说明书
====================================================================

【系统简介】
RaspiClaw 是一个基于 LangChain + LangGraph 的 AI Agent 系统，
支持多模型切换、联网搜索、文件分析、代码执行、记忆管理等能力，
提供现代化的中文 Web 操作界面。

====================================================================

【环境要求】
- Python 3.10 或以上版本
- Windows 10 / 11 操作系统
- DeepSeek API Key（免费注册获取：https://platform.deepseek.com/api_keys）

====================================================================

【安装步骤】

第一步：解压安装包
  将 RaspiClaw-v3.0.5.zip 解压到任意目录（如 D:\RaspiClaw\）

第二步：运行安装脚本
  双击 install.bat，脚本会自动完成：
  1. 检测 Python 环境
  2. 创建虚拟环境（.venv）
  3. 安装所有依赖包（约 2-3 分钟）
  4. 生成配置文件模板

第三步：配置 API 密钥
  用记事本打开 .env 文件，填入你的 DeepSeek API Key：
    DEEPSEEK_API_KEY=sk-你的密钥
  API Key 获取地址：https://platform.deepseek.com/api_keys

第四步：启动系统
  双击 start.bat，约 8 秒后浏览器自动打开。
  默认地址：http://127.0.0.1:8000

====================================================================

【功能说明】

1. 对话聊天
   - 输入框输入消息，Enter 发送，Shift+Enter 换行
   - 支持多会话管理（左侧 + 按钮新建对话）
   - 上方可切换模型商（Provider）和模型等级（Opus/Sonnet/Haiku）

2. 文件操作
   - 点击 📎 上传文件，支持 txt/docx/pdf/xlsx/pptx/py 等格式
   - 点击 📁 设置工作目录，Agent 操作将限制在此目录内

3. 权限模式（重要）
   - 完全权限：Agent 可自由操控电脑文件和命令行
   - 询问权限：每次工具调用需要用户确认（推荐）
   - 安全模式：仅限聊天和数据分析，不能操控电脑

4. 联网搜索
   - 直接输入"搜索xxx"，Agent 自动调用 DuckDuckGo 搜索

5. 记忆管理
   - 左侧"记忆"面板可编辑 Agent 的身份、性格和用户档案

6. 技能系统
   - 左侧"技能"面板查看已加载的技能
   - 支持数据分析（summer.csv）等技能

7. 主题切换
   - 左下角 🌓 按钮切换暗色/亮色界面主题

====================================================================

【添加新的 AI 模型】

1. 进入"设置" → "Provider 管理"
2. 点击 "+ 添加 Provider"
3. 填写信息：
   - 名称：如 "Zhipu GLM"
   - API Key：你的 API 密钥
   - Base URL：API 地址（不带 /v1）
   - 运行时类型：DeepSeek / Qwen / OpenAI兼容
   - Opus/Sonnet/Haiku：分别填对应模型名
4. 保存后在对话页顶部下拉框切换

====================================================================

【常见问题】

Q: 启动提示 "DEEPSEEK_API_KEY not found"
A: 检查 .env 文件是否存在并填入正确的 API Key

Q: 浏览器显示"无法连接"
A: 等待服务器窗口显示 "Application startup complete" 后刷新

Q: 端口被占用
A: 编辑 .env 文件，修改 API_PORT 为其他端口（如 8080）

Q: 依赖安装失败
A: 尝试以管理员身份运行 install.bat
